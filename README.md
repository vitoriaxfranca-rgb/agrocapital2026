# AgroCapital 2026 — Painel de Cobertura

Plataforma de cobertura de evento para a **AgroCapital 2026** (1 a 5 de junho · Goiânia, GO).

## O que é

Painel interativo com:
- Calendário clicável dia a dia (01 a 05 de junho)
- Roteiros completos dos 10 reels (2 por dia)
- Checklist de takes de vídeo por dia
- Checklist de fotos por dia (15 itens/dia)
- Barra de progresso por dia e geral (0–100%)
- Parabenização animada ao concluir reel, dia ou cobertura completa

## Como usar

Acesse via GitHub Pages:

```
https://SEU_USUARIO.github.io/agrocapital2026/
```

Ou abra o arquivo `index.html` diretamente no navegador.

## Deploy no GitHub Pages

1. Suba este repositório no GitHub
2. Vá em **Settings → Pages**
3. Em **Source**, selecione `main` e pasta `/ (root)`
4. Clique em **Save**
5. Aguarde ~1 minuto e acesse a URL gerada

## Estrutura

```
agrocapital2026/
├── index.html   ← toda a plataforma (arquivo único)
└── README.md
```
